// installation

Up in 5 minutes

Download the zip, copy files into your vault, and the wizard handles the rest.

1
Install Dataview plugin
Open Obsidian → Settings → Community Plugins → Browse, search Dataview, install and enable it.
Settings → Community Plugins → Browse
→ search "Dataview" → Install → Enable
→ Dataview Settings → Enable JavaScript Queries ✓
2
Download & copy files
Download the zip, unzip it, copy everything inside zen-os/ into your vault root.
your-vault/
├── Dashboard.md
└── .obsidian/
    ├── plugins/google-workspace-obsidian/
    └── snippets/Dashboard-Zen.css
3
Enable Google Workspace plugin
The plugin is already in the zip — no separate download needed.
Settings → Community Plugins
→ find "Google Workspace" → toggle ON ✓
4
Enable the CSS snippet
This applies the dashboard styling.
Settings → Appearance → CSS Snippets
→ enable "Dashboard-Zen" ✓
5
Open Dashboard.md in Reading View
The 9-step setup wizard launches automatically. Connect Google, pick a theme, add an AI key — done.
Dashboard.md → Reading View
→ wizard launches → follow 9 steps → 🚀
