// installation

Up in 5 minutes

Download the zip, copy files into your vault, and the wizard handles the rest.

1
Install Dataview plugin
In Obsidian → Settings → Community Plugins → Browse, search Dataview, install and enable it. Then turn on Enable JavaScript Queries in Dataview settings.
2
Download & copy files
Download zen-os-v3.0.zip, unzip it, and copy everything inside the zen-os/ folder directly into your vault root.
3
Enable Google Workspace plugin
Settings → Community Plugins → find Google Workspace → toggle on. It's already in the zip — no separate install needed.
4
Enable the CSS snippet
Settings → Appearance → CSS Snippets → enable Dashboard-Zen.
5
Open Dashboard.md in Reading View
The setup wizard launches automatically. Follow the 9 steps — connect Google, pick a theme, add an AI key — and you're done.
